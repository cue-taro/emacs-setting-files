Mongo::getHosts

(PECL mongo >=1.1.0)

Mongo::getHosts — 関連づけられたすべてのホストの状態を更新する

説明

public array Mongo::getHosts ( void )

このメソッドは、レプリカセットへの接続に対してのみ使えます。 レプリカセット内のすべてのホストの状態を返し、接続情報を更新 (ユーザからは見えません) します。

このメソッドは、ドライバが 5 秒おきに自動的にコールします。

読み込みのスレーブへの分散についての情報は、このマニュアルの 問い合わせの節 を参照ください。

パラメータ

この関数にはパラメータはありません。

返り値

セット内のホストに関する情報の配列を返します。 各ホストのホスト名、セット内での id、健康状態 (1 が最高)、稼働時間、 そしてそのホストがプライマリの最新を反映しているかどうかが含まれます。 たとえば、ローカルで稼働しているメンバ 3 台のレプリカセットだとこのようになります。
