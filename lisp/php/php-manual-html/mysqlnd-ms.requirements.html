要件

PHP 5.3.6 以降。

mysqlnd_ms のレプリケーションおよびロードバランシングの プラグインでは、全ての PHP アプリケーションおよび全ての利用可能な PHP MySQL エクステンション (mysqli、 mysql および PDO_MYSQL) をサポートします。 mysqlnd に対して mysqlnd_ms プラグインを使用できるためには、 PHP MySQL エクステンションは mysqlnd_ms を使用して構成されていなければいけません。
