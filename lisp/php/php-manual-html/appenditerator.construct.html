AppendIterator::__construct

(PHP 5 >= 5.1.0)

AppendIterator::__construct — AppendIterator を作成する

説明

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値
