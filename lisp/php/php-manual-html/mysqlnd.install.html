インストール

Unix へのインストール

デフォルトで、MySQL データベース・エクステンションは、 MySQL クライアント・ライブラリを用いて構成されます。 MySQL ネイティブ・ドライバを使用するためには、MySQL データベース・エクステンションを MySQL ネイティブ・ドライバのサポート付でコンパイルすることを指定して PHP をビルドする必要があります。 これは、PHP のソースコードをビルドする前に、構成オプションを通じて行なわれます。

例えば、MySQL ネイティブ・ドライバを用いて MySQL エクステンション、mysqli と PDO MYSQL をビルドするには、以下のコマンドが渡されます。 

./configure --with-mysql=mysqlnd \
--with-mysqli=mysqlnd \
--with-pdo-mysql=mysqlnd \
[other options]

Windows へのインストール

5.3 以降の公式な PHP ディストリビューションでは、 MySQL ネイティブ・ドライバは、デフォルトで使用可能です。 そのため、使用するための追加の構成は不要です。 この場合は MySQLデータベース・エクステンションは全て MySQL ネイティブ・ドライバを使用します。
