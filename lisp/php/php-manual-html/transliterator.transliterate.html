Transliterator::transliterate -- transliterator_transliterate — 文字列を音訳する

説明

オブジェクト指向型

public void Transliterator::transliterate ( string $subject [, string $start [, string $end ]] )

手続き型

transliterator_transliterate ( string $subject [, string $start [, string $end ]] )

文字列を音訳します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

subject

音訳する文字列。

start

end

返り値

成功した場合に音訳された文字列を返します。 失敗した場合に FALSE を返します.
