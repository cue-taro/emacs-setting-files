AMQPQueue::get

(PECL amqp >= Unknown)

AMQPQueue::get — 次のメッセージをキューから取得する

説明

public array AMQPQueue::get ([ int $flags = AMQP_NOACK ] )

次のメッセージをキューから取得します。 これは非ブロック操作です。つまり、もしメッセージがキューになければ、メッセージを取得せずすぐに結果を返します。

パラメータ

flags

フラグ AMQP_NOACK のビットマスク。

返り値

配列を返します。含まれるキーには routing_key, exchange, delivery_tag, Content-type, Content-encoding, type, timestamp, priority, expiration, user_id, app_id, message_id, Reply-to, count, msg といったものがあります。
